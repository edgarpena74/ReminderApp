The require() portion is part of the NodeJS stuff

const express = require("express");
const app = express();

Then the alternative express port is 5000

const PORT = process.env.PORT || 5000;

Then "path" is used for routing. (get, post, put, delete, etc.)

const path = require("path");

DB routing

require("./models/connection");

Handles JSON stuff

app.use(express.urlencoded({ extended: true }));
app.use(express.json());

This is for when we have a build folder.

if (process.env.NODE_ENV == "production") { app.use(express.static("client/build")); }

A custom route for Get, Post, Del, etc.

app.use("/blog", require("./routes/blog-routes"));

Routing for DB stuff

app.get("*", (req, res) => { res.sendFile(path.resolve(__dirname, "client", "build", "index.html")); });

MOST IMPORTANT! The port

app.listen(PORT, () => { console.log("Listening"); });
